Student Fee Tracker

Student Fee Tracker was the final project for my first Java class.

The program is an interactive GUI program that stores and edits student info.

The program allows the user to edit student details like names, addresses, and amounts paid and due, as well as to enter new entries and delete existing ones. It also automatically tracks and displays the total fees paid and due.

This was my first real GUI program and my biggest Java project so far.

The code for this project can be found at my GitHub.
